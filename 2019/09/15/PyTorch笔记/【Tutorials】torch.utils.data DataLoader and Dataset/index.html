PyTorch笔记/【Tutorials】torch.utils.data DataLoader and Dataset

Posted on 2019-09-15 Edited on 2020-03-19 In PyTorch笔记

本节主要讲述PyTorch中的DataLoader与Dataset的使用，并举代码示例用LeNet做人民币的二分类。

图1   获取及处理数据的一般流程

DataLoader

功能: 构建可迭代的数据装载器

  • dataset: Dataset类，决定数据从哪读取以及如何读取
  • batchsize:
  • num_works: 是否采用多进程读取数据，一般会设置为4、6、8
  • shuffle: 每个epoch是否乱序
  • drop_last: 当样本数不能被batchsize整除时，是否舍弃最后一批数据

理解一下 Epoch、Iteration、Batchsize 的概念：\
Epoch: 所有训练样本都已输入到模型中,称为一个Epoch \
Iteration: 一批样本输入到模型中,称之为一个Iteration \
Batchsize: 批大小,决定一个Epoch有多少个Iteration

举个例子：\
样本总数:80, Batchsize:8 \
1 Epoch = 10 Iteration

样本总数:87, Batchsize:8 \
1 Epoch = 10 Iteration ? drop_last = True \
1 Epoch = 11 Iteration ? drop_last = False
