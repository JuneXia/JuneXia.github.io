VSCode字体默认为宋体，难看至极。

File > Preference > Settings > 选中Workspace > 搜索fontFamily > 填入以下内容：
```
'Droid Sans Mono', 'monospace', monospace, 'Droid Sans Fallback',
```

或者，感觉下面这个更好看一些：
```
'Source Code Pro',Consolas,Dengxian
```
